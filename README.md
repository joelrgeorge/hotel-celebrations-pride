## Hotel Celebrations Pride

Welcome to the **Hotel Celebrations Pride** website repository! This is a fully responsive and feature-rich hotel booking site designed to provide an intuitive and seamless booking experience. Built using **React**, the site features a modern UI, dynamic room pricing, and an integrated booking system.
